🔧 DETALHES TÉCNICOS
═══════════════════════════════════════════════════════════════════════════════

Função: handleSaveToSmartDocs()
├─ Valida se existe minuta
├─ Gera nome automático: "tipo_cliente_data.txt"
├─ Auto-detecta tipo (Peticao, Contrato, etc)
├─ Atribui tags (tipo, tom, 'Redator IA')
├─ Calcula tamanho em KB
├─ Codifica conteúdo em base64
├─ Chama addSmartDoc() do Store
├─ Mostra feedback visual de sucesso
└─ Auto-oculta mensagem em 3 segundos

Função: handleExportTxt()
├─ Cria elemento <a> temporário
├─ Define atributo href com conteúdo
├─ Dispara clique para download
├─ Remove elemento do DOM
